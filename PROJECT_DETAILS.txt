Purpose of the project:
This is a personal finance management application designed to help users track their income, expenses, loans, and investments (specifically fixed deposits). It allows users to set monthly budgets, monitor balances, visualize expenses via charts, and manage financial goals like saving through fixed deposits. The app aims to provide a simple, user-friendly interface for financial planning and tracking, promoting better money management.

Features in the project:
- User Authentication: Login page for user access (loginpage.jsx)
- Dashboard (deshbord.jsx):
  - Set monthly income and expenses
  - View total income, balance, and expenses
  - Pie chart for expense breakdown using CanvasJS
  - History of payments (expense table)
  - Fixed deposit reports table
- Fixed Deposit Management (fixeddeposit.jsx):
  - Calculate interest and maturity for fixed deposits (6% rate)
  - Add new FD entries, deduct amount from balance
  - Display FD details in a table
- Loan Management: Track loans (integrated in context, shown in balance calculation)
- Responsive Design: Uses Tailwind CSS for mobile-friendly layout
- Data Persistence: Posts data to local JSON server endpoints for income, expenses, FD, and loans
- Animations: GSAP for button hover effects
- Context-based State: Shared state across components for real-time updates
